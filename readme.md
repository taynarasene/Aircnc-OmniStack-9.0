## Semana Omnisteck

### API Methods

GET, POST, PUT, DELETE

* req.query = Acessar query params (Para Filtros)

```
/users/?idade=25
```
* req.params = Acessar route params (Para Edição e Delete)

```
/users/1
```
* req.body = Acessar o corpo da requisição (Para Criação)

```
{
    "Idade" : 25
}
```
